PROJECT SUMMARY


The purpose of this project was to investigate the relationship between latitude and various weather measurements, such as temperature, cloudiness, wind speed, and humidity. To accomplish this, I used Python and the OpenWeatherMap API to assemble a dataset of over 21,000 cities from around the globe.

Matplotlib and Tableau were used to visualize the data and search for potential correlations. This site provides the source data and visualizations that were generated throughout the analysis, as well as my own opinions as to possible explanations for obeserved trends and correlations.


Drag the slider to the left to see a global heatmap of the raw temperature data for each city, or to the right to see tempeature data normalized to each city's elevation.
